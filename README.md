To run this application please do the following steps:

```npm i -g && npm link```

This is a global setting.

To unlink the application, please give the following command - ```npm unlink create-project```

If you have made any updates just give the following command to update ```npm i -g```

To create a new node.js aws lambda project using this cli, please do the following steps:

1. Go to the directory where you need to create the project
2. Just type ```create-project``` in terminal.
3. You will be given the options to choose TypeScript, to initialize git in the project, to provide a name for the lambda handler and to install the project.
4. Once you have completed all the steps, the node.js aws lambda project will be created for you.
